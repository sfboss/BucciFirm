# BucciFirm Static Site

This repository contains a single-page, static site located at `index.html` in the repo root.

## Quick start

### Option 1: Open directly
Double-click `index.html` to open it in your browser.

### Option 2: Run a local web server (recommended)
From the repo root, run:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.

## Notes
- The page uses CDN-hosted React, Tailwind CSS, and Babel, so an internet connection is required.
